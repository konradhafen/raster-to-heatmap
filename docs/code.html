rastoheat.getCoordinatesOfCellAddress(row, col, geot)[source]¶

Calculate the coordinate pair at the center of a cell address based on the input DEM. UTM projections suggested, will not work with degree units.

Parameters:
  • row – Row index of the raster.
  • col – Column index of the raster.
  • geot – GDAL geotransform for the raster.
Returns:

numpy array with x and y coordinates (array[x, y]).
