Code description¶

rastoheat.ras2heat(raster, outdir)[source]¶

Displays data from an input raster as an interactive web map by converting it to a heatmap and displaying with the Google Maps API

Parameters:
  • raster – input raster
  • outdir – directory to write webpage outputs
Returns:

0 on fail, 1 on success
